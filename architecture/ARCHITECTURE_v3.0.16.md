### 2.2 限流模块架构

#### 2.2.1 令牌桶限流流程

```
┌─────────────────────────────────────────────────────────────────────────────────────┐
│                           令牌桶限流流程 (Token Bucket Rate Limiting)                  │
├─────────────────────────────────────────────────────────────────────────────────────┤
│                                                                                      │
│  请求进入                                                                   限流决策  │
│     │                                                                      │        │
│     ▼                                                                      ▼        │
│  ┌──────────────────────────────────────────────────────────────────────────────┐  │
│  │                         RateLimitMiddleware                                │  │
│  │  ┌──────────────────────────────────────────────────────────────────────┐   │  │
│  │  │  1. 获取客户端标识                                                     │   │  │
│  │  │     ├─ 用户已认证: user:{user_id}                                      │   │  │
│  │  │     └─ 用户未认证: ip:{client_ip}                                      │   │  │
│  │  └──────────────────────────────────────────────────────────────────────┘   │  │
│  │                                      │                                      │  │
│  │                                      ▼                                      │  │
│  │  ┌──────────────────────────────────────────────────────────────────────┐   │  │
│  │  │  2. 获取限流器实例                                                     │   │  │
│  │  │     ├─ 检查 Redis 是否存在分布式限流器                                  │   │  │
│  │  │     │                                                                  │   │  │
│  │  │     │  ┌──────────────────────────────────────────────────────────┐  │   │  │
│  │  │     │  │  Redis Lua Script (原子操作)                             │  │   │  │
│  │  │     │  │  1. 获取当前令牌数 tokens 和最后更新时间 last_update       │  │   │  │
│  │  │     │  │  2. 计算 elapsed = now - last_update                       │  │   │  │
│  │  │     │  │  3. 补充令牌: tokens = min(capacity, tokens + rate * elapsed)│ │   │  │
│  │  │     │  │  4. 尝试消费: if tokens >= requested: tokens -= requested  │  │   │  │
│  │  │     │  │  5. 更新 Redis: tokens, last_update                         │  │   │  │
│  │  │     │  │  6. 返回: [allowed, tokens]                                 │  │   │  │
│  │  │     │  └──────────────────────────────────────────────────────────┘  │   │  │
│  │  │     │                                                                  │   │  │
│  │  │     └─ 本地内存限流器 (无 Redis 时)                                    │   │  │
│  │  └──────────────────────────────────────────────────────────────────────┘   │  │
│  │                                      │                                      │  │
│  │                           ┌──────────┴──────────┐                          │  │
│  │                           │                     │                          │  │
│  │                           ▼                     ▼                          │  │
│  │  ┌────────────────────────────┐    ┌────────────────────────────┐        │  │
```
